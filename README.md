# MansionExplorers Version a.0.0.1
This is a repository for a game in it's earliest stages. Ideas are welcome but please don't report bugs as these are most likely worked on already.

## Description
You and your group of friends find yourselves in a mansion. You don't know anything about this place or what secrets lies ahead.
Explore the mansion as it is built as you play, and uncover the mystery this place holds.

**Feel free to enjoy this work in progress, but keep the version in mind and expect many bugs!**
